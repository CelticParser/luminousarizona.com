## Install

### Install Theme

Extract the themes .zip file to your local computer. Navigate to the project root (it contains the README.md)

Run `npm install` to install node modules.

Then run `npm run start` to start the 11ty server.

To build the 11ty site run `npm run build`

## Deploy

git push --progres

### Netlify

This theme comes with a `netlify.toml` and is tested and configured to deploy with Netlify.
